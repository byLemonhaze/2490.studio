# Portrait 2490 Studio

Collection viewer for *Portrait 2490* — 90 futuristic portraits inscribed on Bitcoin in March 2023, all sub-300k inscriptions.

Live: [2490.studio](https://2490.studio)

---

## What It Is

Portrait 2490 is a time capsule: 90 images of humans and robots living in the year 2490, created at the intersection of early AI image generation and Bitcoin's permanent ledger. Each portrait asks the same question — *what are we gonna look like down the road?* — and answers it with a 2023 perspective locked on-chain forever.

This site is a minimal, immersive viewer for the collection. Slideshow and grid view. No wallet. No marketplace. Just the work.
